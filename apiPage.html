API Information

The Project Morris API provides developers with programmatic access to the data and functionalities of Project Morris. This API is designed to facilitate the retrieval of user and group information related to NSFW content on the Roblox platform.

Base URL

The base URL for all API endpoints is: https://projectmorris.org/api

Endpoints

  • GET /users/<user_id>: Retrieve information about a specific user by their user ID.
  • GET /groups/<group_id>: Retrieve information about a specific group by its group ID.
  • GET /groups>: Retrive all of the archived groups

Response Format

All API responses are returned in JSON format.
